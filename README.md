# Rakamin-x-Kimia-Farma

#Tentang Project
Analisis data yang diberikan mengenai Kimia Farma dari 2020-2023 menggunakan Looker Studio dan BigQuery

#Penjelasan Syntax
Net Sales dan Profit - menghitungkan dan menyajikan transaksi setelah diskon dan laba untung yang didapatkan untuk masing-masing cabang pada tanggal yang ditentukan
Rating Cabang - menghitungkan nilai rata-rata untuk service cabang secara keseluruhan dari 2020-2023

Hasil Syntax tersebut disambungkan ke Looker Studio dengan menyatakan hasil tabel BigQuery sebagai sebuah view yang kemudian disajikan ke Looker Studio untuk visualisasi 
Berikut adalah Komponen yang disajikan dalam dashboard Looker Studio
- Pendapatan Kimia Farma Tahunan
- Top 10 Transaksi dan Profit Provinsi
- Top 5 Cabang Rating dengan Rating Transaction Terendah
- Geo Map Provinsi
